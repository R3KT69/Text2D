# Text2D
Create texts easily with this library! (SDL2.ttf)

## Prerequisites
Need to have SDL2, SDL2.ttf.

## Memory Management? Dont worry!
*It automatically cleans up, dont worry about memory leak!*

Supports any font

## Usage

**First add text.h to your main.cpp file.**
